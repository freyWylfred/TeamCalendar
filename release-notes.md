# Release Notes v1.4.0

## 🎨 デザイン全面リニューアル — "Nordic Slate" テーマ
- **ヘッダー**: Slate-800→900 のグラデーション（深いネイビーで高級感）
- **アクセント**: Indigo-500 (#6366F1) — モダンで洗練された紫青
- **ボタン**: 取得 = Indigo / Excel出力 = Emerald（直感的な色分け）
- **サマリーカード**: ボーダー追加 + フルハイトのアクセントバー
- **グラフ**: インディゴ(会議) + エメラルド(空き) で統一、カードにボーダー
- **タイムライン**: パステル調の行色 (Emerald/Indigo/Amber/Red の -50)
- **テキスト**: Slate-500/800 系で読みやすく統一
- **背景**: 3層の奥行き (Slate-100 → Slate-50 → White)

## 🔧 例外処理の強化
- コンストラクタ / アイコン読込 / テーマ適用 / グラフ描画 / セルダブルクリック等に try-catch 追加
- 設定ファイル I/O のエラーハンドリング強化
- GDI+ リソースリーク修正 (MenuStripRenderer の SolidBrush)

## 📦 配布内容
| ファイル | 説明 |
|---|---|
| `TeamCalendar.exe` | アプリ本体（DLL + ランタイム結合済み） |
| `config.ini` | 勤務時間設定（初回起動時に自動生成） |
| `app.ico` | アプリアイコン |

## 使い方
1. zip を任意のフォルダに展開
2. `TeamCalendar.exe` を実行
3. 必要に応じて `config.ini` で勤務時間を編集
